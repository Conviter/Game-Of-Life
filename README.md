Features:
-Ändern der Updates pro Sekunde während Runtime, von 0 bis 100.
-Hinzufügen von neuen Zellen mit anpassbarem Pinsel oder Fläche mit Linksklick.
-Bestimmung der Dichte der hinzugefügten Zellen.
-rein- und herauszoomen mit dem Mausrad oder dem Slider.
-Bewegen der Kamera mit Rechtsklick
